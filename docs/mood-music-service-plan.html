핵심 가치 제안

화면에 떠다니는 감정 키워드를 2~4개 선택하면, AI가 조합을 분석하여 YouTube Music에서 딱 맞는 플레이리스트를 생성합니다. 앱 설치 없이 토스 안에서 즉시 실행.

타겟 유저

  • 음악은 듣고 싶은데 뭘 들을지 모르겠는 20~30대
  • 기분 전환이 필요한 직장인 (출퇴근, 점심시간)
  • 장르/아티스트 기반 추천에 피로감을 느끼는 유저

앱인토스는 iframe을 금지하지만 YouTube 영상 삽입은 유일한 예외. 이 서비스의 핵심 재생 기능이 플랫폼 정책과 완벽히 맞아떨어짐.

핵심 가정 & 검증 방법

1. 플로팅 버블 UI

가정: 키워드가 떠다니는 인터랙션이 재미있어서 선택 행위 자체가 즐거울 것

검증: 프로토타입 → 5명 사용성 테스트. "3초 안에 뭘 해야 하는지 알겠는가?" 측정

리스크: 모바일 물리 시뮬레이션 성능. TDS 컴포넌트와 조화

2. 감정 조합 추천 엔진

가정: 2~4개 키워드 조합이 단일 키워드보다 만족도 높은 추천을 만들 것

검증: 단일 vs 조합 A/B 테스트. "이 곡이 내 기분에 맞나요?" 5점 척도

리스크: 조합 경우의 수 폭발 → 캐싱 전략 필수

3. YouTube iframe 재생

가정: YouTube 검색 결과가 감정 기반 필터링에 충분한 품질을 제공할 것

검증: 50개 키워드 조합 → 검색 결과 품질 평가 (적합도 70% 이상)

리스크: API quota 일 10,000 units. 백그라운드 재생 제한

핵심 설계 원칙

Spotify는 오디오 피처(수치)로 매칭하지만, YouTube는 텍스트 기반 검색에 의존합니다. 따라서 LLM의 키워드→검색쿼리 변환 품질이 추천 정밀도를 좌우합니다.
